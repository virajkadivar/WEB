(8) General Sibling Selector

The general sibling selector (~) selects all elements that are next siblings of a specified element.

The general sibling selector in CSS selects any element that follows another element. The adjacent sibling selector, however, only selects the element that immediately follows another element. The adjacent sibling selector uses the plus symbol (+).
