(3) :visited Selector

The :visited selector in CSS is used to select the visited links. For example, visit some link on the website and again see it, then it will find the changed color of that link. This change in color is done by :visited the selector//=:visited - This pseudo-class styles links that have been visited.
Syntax:
:visited { // CSS Properties }

Example

(4) :disabled Selector

The :disabled pseudo-class selector provides conditional styling to HTML elements that can receive user input, when the elements have the disabled attribute.
