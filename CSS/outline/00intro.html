outline:-

CSS Outline allows drawing a line outside the border of elements. It is used to set all the properties of the outline in a single declaration.

An outline is a line that's drawn around an element, outside of its border, to make it stand out. Outlines can overlap other content and aren't part of an element's dimensions. They're similar to borders, but they always go around all sides of an element, and they aren't part of the box model, so they don't affect the position of the element or adjacent elements.
