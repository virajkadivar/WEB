HTML Comments

  • HTML comments are ignored by browsers if we comment in HTML code then codes are more readable to us.

    • What are HTML Comments?

  • HTML comments are annotations or notes added in an HTML code that are not displayed by the browser.
  • Just like comments in other languages HTML comments also serve the same purpose of providing code explanations, reminders, or instructions for developers who are collaborating on the same project.

    • Ways to Comment in HTML

  • There are two ways to comment in HTML.

  • Single-line Comment: The single-line comment is given inside the ( <!- comment -> ) tag.
  • Syntax: <!- comment ->

  • Multi-line Comment: It follows the syntax of a single-line comment by adding multiple lines in the comment.
  • Syntax: <!- Multi Line Comment ->

  • Note : The shortcut for adding the comment through the keyboard is by typing Ctrl + / in Windows, and Command + / for Mac users.
